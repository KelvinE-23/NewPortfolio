C# Projects

These projects were built using C#, .NET, ASP.NET MVC, Entity Framework, and core object-oriented programming concepts.

Car Insurance

An ASP.NET MVC web application that mimics a car insurance quote website. It accepts policy information, stores insuree records with Entity Framework Core, calculates quotes using business rules, and includes an admin page for viewing submitted quotes.

Console App Assignment 1

A console application that demonstrates boolean comparisons with loop control. It uses both a while loop and a do while loop to show how conditions determine when code continues or stops.

Math Parameter Console App

A console application that demonstrates method parameters. It defines a method that accepts two integers, performs a math operation on the first number, displays the result, and displays the second number.

Operator Overload

A C# project that overloads the == and != operators for an Employee class. Employee objects are compared by Id, with Equals() and GetHashCode() also overridden for consistent equality behavior.

Polymorphism Assignment

A console application demonstrating abstraction, inheritance, interfaces, and polymorphism. It uses an abstract Person class, an Employee class, and an IQuittable interface to call Quit() through an interface reference.

Student Database Console App

A basic Entity Framework console app with Student and Grade entity classes, a SchoolContext database context, and code that inserts a student record into the database.
